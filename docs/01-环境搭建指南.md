## 2. 每次开机启动集群

```bash
# ===== 步骤1：启动ZooKeeper（三台节点均执行）=====
zkServer.sh start

# ===== 步骤2：启动JournalNode（三台节点均执行）=====
hdfs --daemon start journalnode

# ===== 步骤3：启动HDFS（在 node1 执行）=====
start-dfs.sh

# ===== 步骤4：启动YARN（在 node1 执行）=====
start-yarn.sh

# ===== 步骤5：启动MariaDB =====
systemctl start mariadb

# ===== 步骤6：验证集群 =====
hdfs dfsadmin -report        # 确认DataNode全部在线
yarn node -list               # 确认NodeManager全部RUNNING
mysql -u root -p -e "SELECT VERSION();"  # 确认MariaDB运行
```

## 3. 开发机环境

| 工具 | 版本 | 用途 |
|------|------|------|
| IntelliJ IDEA | 2023+ | Scala/Java IDE |
| JDK | 1.8 | 编译SDK（注意：JDK 21不兼容！） |
| Maven | 3.9+ | 构建工具 |

### IDEA 配置要点

1. `文件 → 项目结构 → SDK` → 添加 JDK 8 路径
2. `文件 → 设置 → Maven` → JDK for importer 选 JDK 8
3. 打开本项目 `pom.xml` 所在的目录即可

## 5. 防火墙配置

如果开发机无法连接 MariaDB（Connection timed out），执行：

```bash
# 开放3306端口
sudo firewall-cmd --add-port=3306/tcp --permanent
sudo firewall-cmd --reload

# 或直接关防火墙（仅实训环境）
sudo systemctl stop firewalld
```

验证：Windows 上 `telnet 192.168.88.101 3306`，出现乱码字符代表连通。
